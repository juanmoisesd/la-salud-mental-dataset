## Descripción

Dataset sobre salud mental en el marco del proyecto "Mental Health and Psychological Resilience in Pandemic Contexts". Incluye datos y estadísticas sobre indicadores de salud mental en contextos de pandemia y situaciones de crisis.

## Overview
This repository contains data and resources related to **la salud mental dataset**. It is part of an open science initiative to share research findings and datasets with the global scientific community.

## Research Context
The project addresses key questions in the field of neuroscience and social sciences, focusing on providing accessible data for further analysis and validation.

## Repository Structure
- `data/`: Contains the datasets used in this research.
- `src/`: Source code for data processing and analysis.
- `results/`: Output files, figures, and metrics.
